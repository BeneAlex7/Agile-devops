# User Stories

## 1. Lister les serveurs (SysOps)
**En tant que** SysOps,
**Je veux** une API REST pour lister les serveurs,
**Afin de** automatiser mes rapports d'infrastructure.

## 2. Optimisation Docker (Développeur)
**En tant que** Développeur,
**Je veux** un Dockerfile multi-stage,
**Afin de** optimiser la taille de l'image finale et réduire les temps de déploiement.

## 3. Tests Automatisés (QA)
**En tant que** QA,
**Je veux** des tests automatisés PyTest,
**Afin de** valider que l'API fonctionne correctement avant chaque mise en production.

## 4. Pipeline CI/CD (DevOps)
**En tant que** DevOps,
**Je veux** un pipeline CI/CD complet,
**Afin de** garantir la qualité du code, la sécurité de l'image Docker, et l'automatisation du déploiement.
